# About

Multi-level slide menu with transactions

# Example
```
<x-slidemenu id="slidemenu" title="All Categories">
	<menu>
	  <li>Clothes
		<menu title="Clothes">
		   <li>Shirts</li>
		   <li>Pants</li>
		   <li>Hats
			 <menu title="Hats">
			   <li>Top Hats</li>
			   <li>BaseBall Caps</li>
			   <li>Bollo Hats</li>
			   <li>Cowboy Hats</li>
			 </menu>
		   </li>
		</menu>
	  </li>
	  <li>Food
		<menu title="Food">
		  <li>Pizza</li>
		  <li>Burritos</li>
		  <li>Sandwiches
			<menu title="Sandwiches">
			  <li>Subs</li>
			  <li>Panini</li>
			  <li>Pita</li>
			  <li>Melts</li>
			</menu>
		  </li>
		</menu>
	  </li>
	  <li>Phones
	   <menu title="Phones">
		  <li>Smartphones</li>
		  <li>Feature Phones</li>
	   </menu>
	  </li>
	</menu>
</x-slidemenu>
```
